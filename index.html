sd-usb-loader by mor

USB Backup/Loader App for Wii.

I'm extending SD/USB Loader 1.5

Install

It's Wii homebrew, you need to put the executable on Wii media and run it with HBC or Loadmii, etc.

Authors

Waninkoko, kwiirk, mor

Contact

mor (xmor61@users.sourceforge.net)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/mor/sd-usb-loader

get the source code on GitHub : mor/sd-usb-loader
